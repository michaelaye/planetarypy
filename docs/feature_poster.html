Auto-Updating Cache

Indexes checked daily, refreshed when stale. Parquet format for fast reads, datetime support, and cloud-readiness.

# Checked daily, refreshed when stale
df = pds.get_index("mro.ctx.edr")

# Force refresh
df = pds.get_index("mro.ctx.edr",
  allow_refresh=True)
🌐

Dynamic URL Discovery

No manual URL tracking. Index URLs discovered automatically by scraping PDS archive pages for the latest volume.

idx.url # auto-discovered
